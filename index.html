Canvas Tool Scripts

Install browser tools for Canvas LMS. Start with Tampermonkey, then install the scripts you need.

Install Tampermonkey
1. Install Tampermonkey

Use the button at the top of this page and install the extension for your browser.

2. Click a script

Each green install button opens Tampermonkey's script install screen.

3. Refresh Canvas

After installing, reload Canvas so the new tools can appear.

Recommended First

Install the dashboard first if you plan to use multiple tools.
